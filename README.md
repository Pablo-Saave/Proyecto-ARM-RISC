# Proyecto-ARM-RISC
# Proyecto ARM y RISC – Simulación en MARS

## Archivos
- `Informe Arquitectura de computadores.pdf`: Documento técnico.
- `Presentación.pdf`: Presentación resumen.
- `src/`: Carpeta con los programas en ensamblador (`secuencial.asm` y `stride.asm`).

## Requisitos
- [MARS 4.5](http://courses.missouristate.edu/KenVollmar/MARS/) (MIPS Assembler and Runtime Simulator).

## Pasos para ejecutar
1.Abre MARS.

2.Presiona F3 (Ensamblar).

3.En el menú de arriba, ve a Tools -> Data Cache Simulator.

4.Dale clic al botón "Connect to MIPS".

5.Dale al botón de "Play" (el triángulo verde) en la ventana principal de MARS.

## Resultados esperados
- **Caso A (Secuencial):** Hit Rate ≈ 74%, menor cantidad de ciclos.
- **Caso B (Stride):** Hit Rate ≈ 0%, mayor cantidad de ciclos.
